# Aspect-Based Sentiment Analysis (ABSA)

This project implements Aspect-Based Sentiment Analysis using:

- SVM with TF-IDF (Baseline)
- Bi-LSTM (Deep Learning)
- BERT Transformer (State-of-the-art)

## Dataset
SemEval 2014 - Restaurant Reviews

## Features
- Aspect-level sentiment classification
- Comparative model evaluation
- Visualization and analysis

## Tech Stack
Python, Scikit-learn, TensorFlow, PyTorch, Transformers
